# 儿童诗词歌赋 - 产品规格

## 1. 项目概述

- **项目名称**: 儿童诗词歌赋
- **类型**: 儿童教育类移动端Web应用
- **核心功能**: 为三岁以下儿童展示精品诗词歌赋，支持滑动浏览
- **目标用户**: 婴幼儿家长、幼教老师

## 2. UI/UX 规格

### 布局结构
- **顶部栏**: 
  - 左侧：名字 + "每日诗词"标签
  - 右侧：模式切换按钮（顺序/随机）
- **主体区域**:
  - 诗题（大字标题）
  - 作者（小字）
  - 诗词正文（大字展示）
- **滑动交互**: 右滑 → 下一篇

### 响应式适配
- 移动端优先，iPhone安全区域适配
- 触摸友好，大字体适合儿童阅读

### 视觉风格
- 背景：柔和的米白色或淡雅中国风
- 字体：清晰易读的中文字体
- 配色：温馨舒适，儿童友好

## 3. 功能规格

### 核心功能
1. **诗词展示**: 50首精选儿童适宜诗词
2. **滑动切换**: 右滑动手势切换诗词
3. **模式切换**: 顺序模式 / 随机模式
4. **名字自定义**: 左上角显示儿童名字

### 数据结构
```json
{
  "poems": [
    {
      "id": 1,
      "title": "咏鹅",
      "author": "骆宾王",
      "content": "鹅鹅鹅，曲项向天歌。\n白毛浮绿水，红掌拨清波。"
    }
  ]
}
```

## 4. 技术栈
- 纯前端：HTML + CSS + JavaScript
- 单页面应用，本地存储

- [ ] 顶部显示名字 + 每日诗词
- [ ] 标题、作者、正文正确显示
- [ ] 右滑动手势切换诗词
- [ ] 右上角模式切换按钮有效
- [ ] 随机模式正常工作
